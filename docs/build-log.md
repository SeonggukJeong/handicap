# Handicap 구현 이력 (build log)

> root `CLAUDE.md`에서 분리된 **과거 이력** — "무엇을 어떻게 만들었나"의 슬라이스/기능별 결과·함정 출처가 여기 모인다. root는 매 세션 필요한 규칙·현재상태 한 줄·크로스커팅 함정·인덱스만 남기고, 이력은 필요할 때 이 파일을 연다.

> 길잡이: **앞으로 할 일** → `docs/roadmap.md` (post-MVP1 단일 진입점). **결정 전문** → `docs/adr/`. **결정 요약 인덱스** → root `CLAUDE.md` `## 알아둘 결정들`. **기능별 상세 작업 메모** → 자동메모리 `MEMORY.md` 항목. **도메인 함정** → 각 디렉토리 중첩 `CLAUDE.md`.
